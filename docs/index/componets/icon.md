## Icon
>ifast的图标使用开源项目 ionicons1.5.2使用<Icon></Icon>组件，指定图标对应的type属性，示例代码：


<Icon type="ion-ionic"></Icon>
<Icon type="ion-ionic" size="20" color="red"></Icon>
<Icon type="ion-ionic" color="#cccc"></Icon>



```html
  <Icon type="ion-ionic"></Icon>
  <Icon type="ion-ionic" size="20" color="red"></Icon>
  <Icon type="ion-ionic" color="#cccc"></Icon>
```
